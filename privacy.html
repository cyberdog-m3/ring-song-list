Privacy

隱私權政策

本頁說明這個非官方粉絲向歌單查詢網站如何處理資料。最後更新：2026-05-28。

我們不主動蒐集個人資料

本網站不提供會員註冊、留言、表單、付款或電子報功能，也沒有自行設定 cookies 或追蹤像素。搜尋與篩選會在你的瀏覽器中執行，不會傳回本網站的伺服器。

網站資料來源

歌曲資料整理自公開 YouTube 直播、公開留言中的時間軸與公開影片資訊。資料僅用於查詢與導向原始 YouTube 內容。

第三方服務

本網站由 GitHub Pages 提供靜態託管，並會載入或連到以下第三方服務：

  • GitHub Pages：提供網站託管，平台可能基於安全目的處理連線紀錄。
  • YouTube / Google：顯示頭像、縮圖並開啟影片連結。
  • Shopee、Discord、Twitter / X：只有在你點擊外部連結後，才會前往對應平台。

外部網站各自適用自己的隱私權政策與服務條款。

資料修正

如果歌名、歌手、時間軸或來源標示有誤，請透過公開社群管道告知。網站維護者會視情況修正資料。
